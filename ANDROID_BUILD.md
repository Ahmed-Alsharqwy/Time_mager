# 📱 NEXUS Android App — Build Guide

## Prerequisites

Install these on your computer before starting:

| Tool | Download |
|------|----------|
| **Node.js 18+** | https://nodejs.org |
| **Android Studio** | https://developer.android.com/studio |
| **JDK 17** | Comes with Android Studio |
| **Git** | https://git-scm.com |

---

## Step 1 — Project Setup

```bash
# Clone / create your project folder
mkdir nexus-app && cd nexus-app

# Copy all files into this folder:
# index.html, manifest.json, sw.js, package.json,
# capacitor.config.json, nexus.css, all .js files,
# icons/ folder (all PNG icons)

# Install Capacitor
npm install
```

---

## Step 2 — Add Android Platform

```bash
npx cap add android
```

This creates an `android/` folder with the full Android project.

---

## Step 3 — Configure Firebase for Android

1. Go to **Firebase Console** → Your Project → Project Settings
2. Click **Add App** → Android
3. Package name: `com.nexus.disciplineos`
4. Download `google-services.json`
5. Place it at: `android/app/google-services.json`

---

## Step 4 — Customize App Info

Edit `android/app/src/main/res/values/strings.xml`:
```xml
<resources>
    <string name="app_name">NEXUS</string>
    <string name="title_activity_main">NEXUS</string>
    <string name="package_name">com.nexus.disciplineos</string>
    <string name="custom_url_scheme">com.nexus.disciplineos</string>
</resources>
```

---

## Step 5 — Sync & Build

```bash
# Sync web files into Android project
npx cap sync android

# Option A: Open in Android Studio (recommended)
npx cap open android
# Then: Build → Generate Signed APK / Bundle

# Option B: Build APK directly
cd android
./gradlew assembleDebug
# APK is at: android/app/build/outputs/apk/debug/app-debug.apk
```

---

## Step 6 — Generate Release APK (for distribution)

In Android Studio:
1. **Build** → **Generate Signed Bundle / APK**
2. Choose **APK**
3. Create a new keystore (save it safely!)
4. Fill in alias, passwords
5. Choose **release** build variant
6. Click **Finish**

APK location: `android/app/build/outputs/apk/release/app-release.apk`

---

## Step 7 — Install on Device

```bash
# Via USB (Developer Mode must be on)
adb install android/app/build/outputs/apk/debug/app-debug.apk

# Or: just copy the APK to your phone and open it
```

---

## Install as PWA (No Build Required)

If you just want to install from Chrome browser on Android:

1. Host the files on GitHub Pages or any server
2. Open the site in Chrome on Android
3. Tap the **⋮ menu** → **"Add to Home Screen"**
4. The app installs like a native app — works offline!

---

## PWA vs APK Comparison

| Feature | PWA (Chrome) | APK (Capacitor) |
|---------|-------------|-----------------|
| Installation | From browser | From APK file |
| Offline | ✅ | ✅ |
| Push notifications | ✅ | ✅ |
| Home screen icon | ✅ | ✅ |
| Splash screen | Basic | Custom ✅ |
| Status bar control | No | ✅ |
| Back button | Limited | ✅ |
| Wake Lock | ✅ | ✅ |
| Firebase Auth | ✅ | ✅ |
| Google Play | No | ✅ |
| No browser UI | Partial | ✅ Full |

## Common Issues

**"INSTALL_FAILED_INSUFFICIENT_STORAGE"**
→ Free up storage on your phone

**"App not installed" error**
→ Enable "Install unknown apps" in Settings → Security

**Firebase auth not working in APK**
→ Add `com.nexus.disciplineos` to Firebase → Authentication → Authorized domains

**White screen on launch**
→ Check that `webDir` in `capacitor.config.json` points to the correct folder

## Quick Commands Reference

```bash
npm install                  # Install dependencies
npx cap add android          # Add Android platform
npx cap sync android         # Sync web files → Android
npx cap open android         # Open in Android Studio
npx cap run android          # Build + run on connected device
./gradlew assembleDebug      # Build debug APK (in android/ folder)
./gradlew assembleRelease    # Build release APK
adb install app-debug.apk   # Install on device via USB
```
